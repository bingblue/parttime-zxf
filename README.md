## 使用方法

```cmd
// 安装依赖
$ cnpm i

// 运行项目
$ npm start

// 替换HTML内容
$ npm run html //如:head.html代码在part文件夹下，运行此命令会替换所有html的头部代码

// 删除替换HTML内容,删除生成文件
$ npm run del
```
